Privacy Policy for Mio Bookmark

Effective date: 2026-03-15

Mio Bookmark respects your privacy. This Privacy Policy explains what information is collected, how it is used, and how it is protected when you use the Mio Bookmark Chrome extension.

1. Purpose of the Extension

Mio Bookmark is designed solely to help users manage Chrome bookmarks in a more visual and convenient way. The extension allows users to browse bookmarks in a folder tree, search quickly, reorganize bookmarks, save the current page, sync bookmarks from the current Chrome profile, and customize folder display with icons, colors, and interface preferences. All features exist only to support organizing, accessing, and personalizing bookmarks within the browser.

2. Information We Collect

We may collect the following information only when necessary to provide the service:

  • Email address provided by the user for sending an activation key or enabling Premium access.
  • Bookmark-related data already available in the user's Chrome profile, only as required for the extension's core bookmark management features.
  • Customization preferences such as icons, colors, layout settings, and interface choices used to personalize the bookmark experience.

3. How We Use the Information

We use the collected information only for the following purposes:

  • To provide the extension's bookmark management features.
  • To display, organize, search, edit, and personalize bookmarks inside Chrome.
  • To sync and maintain bookmark-related settings for the current user profile.
  • To send activation keys or verify Premium access when the user provides an email address.
  • To provide support related to access or service functionality.

4. Data Sharing

We do not sell personal information. We do not share user data with third parties except when necessary to operate the service, such as using an email delivery provider to send activation emails, or when required by law.

5. Data Retention

We retain personal information only for as long as necessary to provide the service, comply with legal obligations, resolve disputes, and enforce our agreements.

6. Data Security

We take reasonable measures to protect user information against unauthorized access, alteration, disclosure, or destruction.

7. Your Choices

You may contact us to request access, correction, or deletion of your personal information, subject to applicable law.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date.
