1. Purpose of the Extension

Mio Bookmark is designed solely to help users manage Chrome bookmarks in a more visual and convenient way. The extension lets users browse bookmarks in a folder tree, search quickly, reorganize items, save the current page, sync bookmarks from the current Chrome profile, and customize folder display with icons, colors, and interface settings.

All features are intended only to support organizing, accessing, and personalizing bookmarks within the browser.

2. Information We Collect

We collect or access information only when necessary to provide the extension's functionality.

  • Email address: only when the user voluntarily provides it to receive an activation key or enable Premium access.
  • Bookmark-related data: bookmark folders, bookmark items, and related structure available in the user's Chrome profile, only as needed for bookmark management features.
  • Customization preferences: icons, colors, layout settings, and interface preferences chosen by the user.
Mio Bookmark does not collect personal information beyond what is necessary to provide bookmark management and optional Premium activation.

3. How We Use Information

  • To provide the extension's core bookmark management features.
  • To display, organize, search, edit, and personalize bookmarks in Chrome.
  • To sync and maintain bookmark-related settings for the current user profile.
  • To send activation keys or verify Premium access when a user provides an email address.
  • To respond to support requests related to extension access or functionality.

4. Data Sharing

We do not sell personal information. We do not share user data with third parties except when necessary to operate the service, such as using an email delivery provider to send activation emails, or when required by law.

5. Data Retention

We retain personal information only for as long as necessary to provide the service, comply with legal obligations, resolve disputes, and enforce our agreements.

6. Data Security

We take reasonable administrative, technical, and organizational measures to protect information against unauthorized access, alteration, disclosure, or destruction.

7. Your Choices
